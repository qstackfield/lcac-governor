    API Reference

    LCAC API Reference Atom Labs · 2025

    ⸻

    Base URL

    https://api.atomlabs.app

    All endpoints are JSON-based and HTTPS-only.

    ⸻

    Authentication

    Free Tier

    If no license key is provided, LCAC automatically issues a trial key on first /evaluate.

    Licensed Tier

    Pass your key via header:

    X-License-Key:

    ⸻

    Endpoints

    ⸻

    1. POST /evaluate

    Runs a full cognitive integrity evaluation.

    Request

    { "prompt": "Explain quantum gravity to a child.", "output": "It is like magic glue in space." }

    Response

    { "license": "trial_129ab3ac", "trace_id": "d511ede8-39cd-4694-8f48-32a65d36a91f", "trust": 0.612, "variance": 0.004, "verdict": "unstable", "recommendation": "Cognitive drift detected. Reduce context overlap.", "severity": { "level": 3, "label": "critical" }, "insight": "LCAC observed reasoning state 'UNSTABLE'..." }

    ⸻

    1. GET /overview

    Unified cognitive state.

    Response

    { "trust": 0.598, "variance": 0.002, "verdict": "unstable", "mode": "HOLD", "persona": "default", "insight": "Monitoring cognitive drift.", "ledger_count": 59 }

    ⸻

    1. GET /metrics

    Provides incremental trust/variance metrics for dashboards.

    Response

    { "ts": "2025-11-14T05:01:12Z", "trust_score": 0.607, "variance": 0.001, "stability": 0.999, "ledger_count": 22, "summary": "UNSTABLE • trust=0.607 var=0.001" }

    ⸻

    1. GET /info

    System and runtime metadata.

    Response

    { "product": "LCAC Cognitive Integrity Framework", "version": "1.0.0-Insight", "platform": "Linux", "mode": "HOLD", "trust_score": "0.607", "uptime_sec": 12402.4, "cpu": 12.4, "memory": 22.7, "prices": { "starter": "price_XXXX", "pro": "price_YYYY", "enterprise": "price_ZZZZ" } }

    ⸻

    1. GET /license/verify

    Validate a key or retrieve metadata.

    Request

    GET /license/verify?key=trial_abc123

    Response

    { "ok": true, "license": { "tier": "trial", "quota": "1000", "used": "22", "status": "active" } }

    ⸻

    1. POST /stripe/checkout

    Creates a Stripe checkout session.

    Request

    { "tier": "starter", "email": "user@example.com" }

    Response

    { "url": "https://checkout.stripe.com/c/session/abc123" }

    ⸻

    1. POST /stripe/webhook

    Used internally by Stripe to issue licenses. Not used by end-users directly.

    ⸻

    Error Formats

    All LCAC errors follow:

    { "error": "message" }

    Quota errors:

    { "detail": "Quota exceeded. Purchase tokens to continue." }

    Invalid or expired license:

    { "detail": "Invalid or expired license key" }

    ⸻

    Rate Limits

    LCAC applies soft rate limits: • Trial: 1,000 evals / 7 days • Starter: 5,000 evals / pack • Pro: 25,000 evals / month • Enterprise: negotiable

    ⸻

    Recommended Headers

    Content-Type: application/json X-License-Key:

    ⸻

    That’s it.

    Minimal, stable, clean.
